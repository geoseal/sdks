# Security Policy

## Reporting a vulnerability

Please report security issues privately to **hello@geoseal.dev**. Do not open a public issue for security reports.

Include the affected package and version, reproduction steps, and impact where possible. We will acknowledge your report and follow up by email.

## Scope

This repository is the public source mirror for the published `@geoseal/*` npm packages and the `CheckpointCore` CocoaPod. Reports about the Geoseal API and platform (geoseal.dev) are also welcome at the same address.
